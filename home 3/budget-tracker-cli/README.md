# Budget Tracker CLI

Утилита командной строки для анализа бюджета с набором вспомогательных функций для работы с числами и строками.

## Функциональность

### Основные функции:

1. **calculateTotal** - вычисляет сумму массива чисел
2. **calculateAverage** - вычисляет среднее значение массива чисел
3. **formatCurrency** - форматирует число в строку валюты
4. **getTopValues** - возвращает N наибольших значений из массива
5. **printSummary** - выводит сводку по массиву чисел

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```bash

2. Скомпилируйте TypeScript:
```bash
npm run build
```bash

3. Запустите приложение:
```bash
npm start
```bash

    Или одной командой:
```bash
npm run dev
```bash
