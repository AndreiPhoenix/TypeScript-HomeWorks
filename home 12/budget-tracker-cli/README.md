# Budget Tracker CLI

Приложение для учета личных финансов с использованием TypeScript и декораторов.

## Возможности

- Управление счетами и транзакциями
- Логирование операций через декораторы
- Защита свойств от изменений
- Работа с метаданными
- Экспорт данных

## Установка
```
npm install
```

## Сборка
```
npm run build
```

## Запуск
```
npm start
```

## Разработка
```
npm run dev
```
