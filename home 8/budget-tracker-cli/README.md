Выполните команды в терминале:

# Установка зависимостей
npm install

# Установка типов для сторонних библиотек
npm install --save-dev @types/uuid @types/node

# Установка библиотек
npm install uuid moment

Запуск приложения:
# Запуск в режиме разработки
npm start

# Или сборка проекта
npm run build

# Запуск собранного проекта
node dist/index.js
